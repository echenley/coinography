# Coinography

This is a demo React Native app built with [Expo](https://expo.io/), [Victory Native](https://github.com/FormidableLabs/victory-native), and the [GDAX API](https://docs.gdax.com/).

## Running Locally

```
# using yarn
yarn && yarn start

# using npm
npm i && npm start
```
